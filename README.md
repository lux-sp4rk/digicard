# DigiCard

**Live:** [luxsp4rk.com](https://lux-sprwhk.xyz/)

A personal portfolio that earns its name — dynamic theming, CMS-driven content, and integrations with the creator stack (Beehiiv, YouTube, Instagram). No page rebuilds to change the look.

## What it is

An [Astro](https://astro.build) single-page portfolio with React islands, five switchable themes, and Contentful CMS backing. Each theme is a full visual overhaul — not just a color swap. Content (projects, services, skills) is editable in Contentful and falls back to static data when the CMS is unavailable.

**Themes:** Catppuccin Mocha · Flexoki Light · Web2.0 · Matrix

## Features

- **Dynamic theming** — four themes, switchable in real time, each a full visual overhaul
- **Contentful CMS** — all content is editable without code; static fallback when API keys aren't present
- **Creator integrations** — latest YouTube video, latest Instagram post, Beehiiv newsletter CTA
- **Responsive** — works across desktop and mobile
- **Console easter eggs** — because boring portfolios are forgettable

## Stack

Astro · React · Tailwind CSS · Contentful · Netlify
